# Álgebra Linear e Geometria Analítica | Python

Este repositório integra conceitos de Álgebra Linear e Geometria Analítica com a prática computacional em Python. O objetivo principal consistiu em utilizar a programação como ferramenta de apoio à engenharia, automatizando cálculos matemáticos fundamentais e otimizando a resolução de problemas complexos.

## O que foi desenvolvido?
O projeto abrange a implementação de algoritmos para a manipulação e operação de matrizes e vetores. A abordagem computacional incluiu tanto o desenvolvimento de lógicas manuais, utilizando estruturas de dados nativas do Python (listas de listas), quanto a aplicação de bibliotecas consagradas no mercado para análises numéricas e científicas.

## Tecnologias e Conceitos Aplicados
A estruturação do código foi desenhada para garantir eficiência matemática e modularidade. As principais competências e ferramentas trabalhadas incluem:

* **Computação Científica:** Utilização avançada da biblioteca NumPy e do módulo Math para o processamento otimizado de operações matemáticas.
* **Operações Matriciais e Vetoriais:** Implementação de funções matemáticas para soma e multiplicação de matrizes, cálculo de norma vetorial, produto escalar, determinante e verificação de ortogonalidade.
