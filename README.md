Este repositório é dedicado ao estudo de Geometria Analítica e Álgebra Linear com o uso de Python, reunindo implementações práticas para facilitar a compreensão dos conceitos vistos em aula. O projeto foi desenvolvido com foco no aprendizado, utilizando programação como ferramenta de apoio à matemática.

Os códigos abordam tópicos como vetores, matrizes, sistemas lineares, retas, planos, cônicas e transformações geométricas, além de visualizações em 2D e 3D. São utilizadas bibliotecas como NumPy, SymPy e Matplotlib para cálculos e representações gráficas.

O objetivo do repositório é consolidar os conhecimentos teóricos por meio da prática computacional, servindo como material de estudo e apoio para disciplinas de matemática e áreas relacionadas à tecnologia.
