# Algebra_Geometria_python
Este repositório é dedicado ao estudo e à aplicação de conceitos de Geometria Analítica e Álgebra Linear utilizando Python. O objetivo é fornecer uma base prática e interativa para estudantes, professores e entusiastas de matemática, permitindo compreender melhor os conceitos teóricos por meio de implementações computacionais.

Na parte de Geometria Analítica, o repositório aborda o estudo de pontos, distâncias, segmentos, retas, planos e ângulos, além de cônicas como circunferência, parábola, elipse e hipérbole. Também inclui transformações geométricas como translação, rotação e reflexão, e cálculos envolvendo produto escalar, vetorial e misto.

Na parte de Álgebra Linear, são abordados vetores, operações entre vetores, matrizes, determinantes, inversas, transposição e sistemas lineares. O repositório também cobre conceitos avançados como espaços vetoriais, subespaços, autovalores, autovetores, diagonalização e aplicações em transformações lineares.

O projeto utiliza bibliotecas Python como NumPy, SymPy, Matplotlib, Plotly e SciPy, permitindo tanto cálculos matemáticos precisos quanto visualizações em 2D e 3D. Além disso, contém notebooks interativos e scripts prontos para estudo e experimentação, facilitando o aprendizado e a aplicação prática dos conceitos.

Este repositório é uma ferramenta completa para quem deseja consolidar conhecimentos em Geometria Analítica e Álgebra Linear, com ênfase em programação, visualização e aplicações reais.
