Este repositório é dedicado ao estudo de Geometria Analítica e Álgebra Linear com o uso de Python, reunindo implementações práticas para apoiar a compreensão dos conteúdos matemáticos vistos em aula. Além dos conceitos teóricos, os códigos trabalham fundamentos básicos de programação, como variáveis, tipos de dados, operações matemáticas, estruturas condicionais, laços de repetição, funções e manipulação de vetores e matrizes.

O projeto utiliza bibliotecas como NumPy, SymPy e Matplotlib para cálculos e visualizações, com o objetivo de consolidar o aprendizado por meio da prática computacional e servir como material de estudo para disciplinas de matemática e áreas relacionadas à tecnologia.
